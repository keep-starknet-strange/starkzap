## Installation

Install Starkzap using npm or yarn:

```bash
npm install @starkware-ecosystem/starkzap
```

or

```bash
yarn add @starkware-ecosystem/starkzap
```

### Dependencies

The SDK has two peer dependencies:

- [`starknet`](https://www.npmjs.com/package/starknet) (v9+) - Starknet.js core library (installed with `@starkware-ecosystem/starkzap`)
- [`@cartridge/controller`](https://www.npmjs.com/package/@cartridge/controller) - optional peer dependency, only needed for Cartridge support

These will be installed automatically when you install `@starkware-ecosystem/starkzap`.
